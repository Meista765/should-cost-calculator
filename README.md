# 프로젝트 목적
판금 프레스 3D 설계 변경에 따른 가격 AS-IS, TO-BE 계산기

# 데이터
## 사용자 입력란
### 원소재
- 폭(mm)
- 피치(mm)
- 두께(mm)
- 스크랩 회수율(%): Default = 90%
### 제품
- 너비(mm)
- 길이(mm)
- 높이(mm)
- 표면적(mm²)
- 체적(mm³)
### 설비
- 총 공정 수
  - 드롭다운 형식(1~20)
  - Default = 1
- 시간당 생산 수량(UPH; Unit Per Hour, EA/hr)
  - 프로: Default = 720EA/hr
  - 단발: Default = 180EA/hr

## DB에서 Query
### 원소재
- 강종: DB에서 드롭다운 형식으로 선택
- 비중(g/cm³)
- 원코일 KG당 가격(KRW/kg)
- 스크랩 KG당 가격(KRW/kg)
### 설비
- 구분: DB에서 드롭다운 형식으로 선택
- 톤수: DB에서 드롭다운 형식으로 선택
- 설비임율(KRW/hr)
- 노무임율(KRW/hr)

## 자동 계산
- 원소재 중량(kg) = 폭 * 피치 * 두께 * 비중 / 10^6
- 제품 중량(kg) = 체적 * 비중 / 10^6
- 스크랩 중량(kg) = (원소재 중량 - 제품 중량) * 스크랩 회수율
- 재료비(KRW/EA) = 원소재 중량 * 원코일 KG당 가격 - 스크랩 중량 * 스크랩 KG당 가격
- 가공비(KRW/EA) = 개별 가공비의 총합
  - 사용자가 입력한 총 공정 수를 바탕으로 입력 행 생성
  - 구분, 톤수에 따라 DB에서 설비임율을 불러오고,
  - DB에서 노무임율은 절단원으로 불러올 것.
  - 계산식: (설비임율 + 노무임율) / UPH

# 서비스
## CASE 1: 사용자가 AS-IS 정보 입력 유무와 무관하게 항상 표시
1. 동일 강종 사양으로 두께가 변경되었을 때 TO-BE 가격 표시
  - 원소재 변경 시 체적 추산 = 제품 체적 / 변경 전 두께 * 변경 후 두께
  - 변경된 원소재비를 바탕으로 사용자가 AS-IS로 입력했던 정보 대비 원소재 증감 폭을 주식 증감처럼 표시

2. 동일 두께 사양으로 원소재가 변경되었을 때 TO-BE 가격 표시
  - 현재 사용자가 AS-IS에 입력한 정보를 바탕으로 강종을 변경했을 때의 가격 표시

## CASE 2: 사용자가 AS-IS 정보를 입력했을 때 표시 (Optional)
1. 변경 전,후 제품 사양, 설비 사양을 입력하면, 아래 두 가지 항목으로 분할하여 사용자에게 표시
   - 재료비는 얼마나 바뀌는지
   - 가공비는 얼마나 바뀌는지
